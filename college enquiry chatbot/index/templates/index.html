    About Us



    This project aims to develop a college enquiry Chabot that answers any queries post by students like college details, course-related questions, fee structure etc. This System is a web application that provides answers to the query. Any individual just has to query through the bot. The answers are appropriate to what the user queries. The User can query any college-related activities through the system. The user does not have to personally go to the college for enquiry. The System analyses the question and then answers to the user. The user can also give their suggestions through the suggestion box. The system replies using an effective Graphical User Interface which implies that as if a real person is talking to the user.
